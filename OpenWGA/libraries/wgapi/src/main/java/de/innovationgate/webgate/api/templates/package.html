Templates to create your own simple content/design/repository sources. The main classes inside here are:

{@link de.innovationgate.webgate.api.templates#SimpleContentSource SimpleContentSource}

The base class for all other content-based template classes. Provides limited read/write functionality.

{@link de.innovationgate.webgate.api.templates#QueryableSource QueryableSource}

A very basic template to provide access to query-only sources.

{@link de.innovationgate.webgate.api.templates#SimpleFileRepository SimpleFileRepository}

This is a template to create your own file repositories serving files.
